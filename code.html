R Packages

Programming with Big Data in R (pbdR)

For more information about the pbdR project, see http://r-pbd.org/

pbdPAPI

The most advanced profiling suite available for R. Measures flops, cache misses, cpu utilization, and much more.

pbdPROF

A package that greatly simplifies the use of MPI profilers with R. In addition to easing the burden of building and linking select MPI profilers, the package provides utilities for parsing, analyzing, and plotting profiler outputs.

pbdDEMO

Over 30 demonstrations and examples of high-performance computing with R and pbdR. Also includes a 130+ page, textbook-style vignette that helps transition desktop R users into being able to leverage multi-node compute resources.

pbdDMAT

High-level programming methods for large scale, distributed programming in R. Includes numerous linear algebra and statistics methods for distributed matrices. Benchmarked to 50,000 cores, but can achieve speedups on smaller servers and workstations.

pbdBASE

Interface to ScaLAPACK distributed linear algebra routines plus hundreds of helper routines; uesd by pbdDMAT.

pbdMPI

Efficient, simplified interface to MPI for R.

pbdinline

An R package to emulate interactivity with (batch) pbdR.

System Hardware Info for R and Analytics (SHINRA)

For more information about SHINRA, see https://shinra-dev.github.io/

memuse

A package for benchmarkers. Estimates the memory usage of matrices and other objects. Contains numerous other convenience utilities, including those for gathering hardware RAM, swap, and cache information.

merkhet

Contains utilities for times, including queries for system and process uptimes, as well as a post-hoc system.time().

okcpuid

R package that gives some basic cpu information (clock speed, number of cores, etc.). Also includes a custom LINPACK-like benchmarker.

scribe

A set of enhancements and visualizations for the hardware counter profiler pbdPAPI.

stressR

is a suite of tools for stressing hardware, including cpu, memory, and disk.


Other HPC Packages

pcapack

An experimental package for computing principal components and related statistical quantities.

linmod

An experimental rewrite of R's lm.fit() and glm.fit() functions. Like R's implementation, it uses the "limited pivoting" strategy in a rank-revealing QR; unlike R, we use level 3 BLAS (and generally better memory management) when we can.

Rth

Multicore programming in R through the thrust template library. Compiles to an OpenMP, Intel Thread Buildingblocks, or CUDA backend.

rexpokit

A set of bindings and utility functions for computing matrix exponentials via the Fortran 77 package EXPOKIT.


Utilities

dequer

A package offering a custom data structure for R. A deque is like a list, but with very cheap insertions (since R lists are contiguous arrays of pointers).

lineSampler

Samples a file by line; quickly read a subsample of observations from a large flat text file (e.g., a csv). Useful for downsampling; not suitable for resampling.

RNACI

A collection of utilities that make dealing with R's native C interface more manageable. Notably the package handles simplified SEXP allocation (with automatic gc counting), and the creation and management of lists and dataframes. For instance, by making use of C99's va_args, we can create arbitrary lists of named objects in just 2 simple function calls.

yalda

"Yet Another LDA" package. Wraps the serial portion of the Google plda program for use with R.

rgutenberg

A resource for those who need quick access to some sample text corpora. Just as R's rnorm() can be used to generate "fake" numerical data, the rgutenverg package's rgutenberg() samples a random book from Project Gutenberg. The package also includes numerous texts as easily accessed, already preprocessed datasets.

f2cp

A set of parsers to scan Fortran source files and generate C prototypes.


Just for Fun

Rdym

A "did you mean?" tool for R to help catch spelling mistakes in an interactive R session.

RTop500

A package containing the Top 500 supercomputer data, as well as some utilities for plot generation.

timeRtypeR

A utility that measures your typing speed in an R terminal. Note that it breaks a lot of the nice features of the R terminal (like, say, the ability to backspace).

hackR

An R package for translating text into "leet speak", or for generating a goofy circa 1990's hacker handle.

Rfiglet

R bindings for figlet; creates ascii logos. See also Figlet as a Service.

xkcdpw

A package for making portmanteau-style passwords, as in this famous XKCD comic. Words are taken from scrapings of random Wikipedia pages.

tfca

Some methods for computing transfinite cardinal arithmetic.

gmhelper

A collection of utilities for game masters of fantasy-based role playing games, with a GUI and everything!

idgaf

All public GitHub commits containing the word "fuck" in the commit message, from 3/11/2012 to 7/24/2014.



Libraries

libexpm

High-performance C library for computing the matrix exponential of a dense matrix.

meminfo

A fully cross-platform C library for accessing basic hardware information, such as total RAM, free RAM, total swap, etc. Additionally, can get cache sizes and cache line sizes. Supports Linux, Windows, Mac OS X, and FreeBSD. Used by the R package "memuse".

ngram

A C library for reading, processing, and "babbling" n-gram models. Used in the R package "ngram".



Webapps

hackR

An interactive interface to the hackR package. Now you can hack the gibson without needing to learn R!

ngram

An interactive interface to the ngram package. Ever wanted to make your very own wacky sentences using markov chains? Now you can!

RTop500

A visual summary of the performance over time (by various metrics) of the top 500 supercomputers since 1993. Written with R and shiny. Source is available in the RTop500 package.

Figlet as a Service

A demo of the Rfiglet package. Written with R and shiny. Source is available in the Rfiglet package.
