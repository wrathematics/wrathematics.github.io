Below are some selected projects for which I have a lead authorship. Things to which I have merely contributed to are not listed.

R packages

Utilities

memuse: A package for benchmarkers. Estimates the memory usage of matrices and other objects. Contains numerous other convenience utilities, including those for gathering hardware RAM, swap, and cache information.

Rth: Multicore programming in R through thrust

RNACI: A collection of utilities that make dealing with R's native C interface more manageable. Notably the package handles simplified SEXP allocation (with automatic gc counting), and the creation and management of lists and dataframes. For instance, by making use of C99's va_args, we can create arbitrary lists of named objects in just 2 simple function calls.

yalda: "Yet Another LDA" package. Wraps the serial portion of the Google plda program for use with R.

cpuid: R package that gives some basic cpu information (clock speed, number of cores, etc.).

rgutenberg: A resource for those who need quick access to some sample text corpora. Just as R's rnorm() can be used to generate "fake" numerical data, the rgutenverg package's rgutenberg() samples a random book from Project Gutenberg. The package also includes numerous texts as easily accessed, already preprocessed datasets.

f2cp: A set of parsers to scan Fortran source files and generate C prototypes.



Just for Fun

Rfiglet: R bindings for figlet. Creates ascii logos.

xkcdpw: A package for making portmanteau-style passwords, as in this famous XKCD comic. Words are taken from scrapings of random Wikipedia pages.

tfca: Some methods for computing transfinite cardinal arithmetic.

gmhelper: A collection of utilities for game masters of fantasy-based role playing games, with a GUI and everything!

idgaf: All public GitHub commits containing the word "fuck" in the commit message, from 3/11/2012 to 7/24/2014.



Programming with Big Data in R (pbdR)

pbdPAPI: The most advanced profiling suite available for R. Measures flops, cache misses, cpu utilization, and much more.

pbdDEMO: Over 30 demonstrations and examples of high-performance computing with R and pbdR.

pbdDMAT: High-level programming methods for large scale, distributed programming in R. Includes numerous linear algebra and statistics methods for distributed matrices. Benchmarked to 50,000 cores, but can achieve speedups on smaller servers and workstations.

pbdBASE: Interface to ScaLAPACK distributed linear algebra routines plus hundreds of helper routines; uesd by pbdDMAT.

pbdPROF: A package that greatly simplifies the use of MPI profilers with R. In addition to easing the burden of building and linking select MPI profilers, the package provides utilities for parsing, analyzing, and plotting profiler outputs.

pbdMPI: Efficient, simplified interface to MPI for R.

pbdinline: An R package to emulate interactivity with (batch) pbdR.



Libraries

meminfo: A fully cross-platform C library for accessing basic hardware information, such as total RAM, free RAM, total swap, etc. Additionally, can get cache sizes and cache line sizes. Supports Linux, Windows, Mac OS X, and FreeBSD.

ngram: A C library for reading, processing, and "babbling" n-gram models. Used in the ngram R package.



Training, Documentation, and Presentations

installation-instructions: A comprehensive set of instructions for installing and maintaining an MPI library as well as the pbdR packages.

pbd-tutorial: Sets of slides and examples from several pbdR tutorials.

UseRdocs: My presentations/tutorials/posters given at the R User conference.

FortranTutorial: A beginner's guide to the Fortran programming language.
